# fluent-plugin-syscheck

[Fluentd](https://fluentd.org/) input plugins to check system parts.



## Installation

### RubyGems

```
$ gem install fluent-plugin-syscheck
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-syscheck"
```

And then execute:

```
$ bundle
```
